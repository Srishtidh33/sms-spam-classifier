# AI SMS Spam Detector 📩

This project is a **Machine Learning based SMS Spam Detection system** that classifies messages as **Spam or Not Spam**.

The model is trained on a dataset of SMS messages and uses **TF-IDF vectorization with a Naive Bayes classifier**.

## 🚀 Live Demo

https://sms-spam-classifier-ai.streamlit.app/

## 📌 Features

* Detects spam SMS messages
* Simple web interface
* Machine learning based prediction
* Real-time message classification

## 🧠 Technologies Used

* Python
* Scikit-learn
* Pandas
* Streamlit
* TF-IDF Vectorization

## ▶️ Run Locally

Install dependencies:

pip install -r requirements.txt

Run the web app:

streamlit run app.py

## 📊 Model

Algorithm used: **Multinomial Naive Bayes**
